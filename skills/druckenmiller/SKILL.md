# Druckenmiller Macro

Use this skill to apply Stanley Druckenmiller-style macro judgment: identify the dominant factor, align with liquidity and market confirmation, concentrate when the setup is exceptional, and cut losses quickly when wrong.

## When To Use

Use this skill when the user asks for:

- Top-down macro trade analysis
- Liquidity and central-bank regime review
- Currency, rates, equity index, or commodity theme analysis
- Whether to press a high-conviction winner
- Risk management for concentrated macro positions

Trigger phrases include `Druckenmiller`, `macro`, `liquidity`, `central bank`, `rates`, `currency`, `press winners`, `concentrated trade`, and `cut losses`.

## Do Not Use When

- There is no clear dominant macro driver.
- The trade has no liquid expression.
- The user wants a low-volatility diversified allocation.
- The setup is mediocre but the user wants oversized risk.

## Inputs Needed

- Macro theme or asset
- Central-bank, liquidity, rates, currency, growth, and inflation context
- Price trend and cross-asset confirmation
- Consensus positioning and expected catalyst path
- Current exposure and loss limit if sizing is requested

## Process

1. Identify the single dominant driver.
2. Check whether liquidity, policy, and price action align.
3. Compare the thesis to consensus and positioning.
4. Define the cleanest liquid expression.
5. Start with risk that can be cut quickly.
6. Press only if the thesis is working and confirmation improves.
7. Exit fast if the market rejects the thesis.


## Guardrails

- Do not diversify away the edge when the setup is exceptional.
- Do not concentrate in an unconfirmed or illiquid setup.
- Do not let a losing macro view become an identity.
- Do not fight price action indefinitely.
- Cut quickly when the thesis is wrong.
