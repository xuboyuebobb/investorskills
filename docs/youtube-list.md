# Investor Skills YouTube List

Recommended YouTube videos and channels mapped to skills in this repo.

This file is **reference material for the skills**, not a replacement for them.

## How To Use As Skill Reference Material

1. Load `skills/<name>/SKILL.md` and `invest.md` first.
2. Watch 1 mapped video/channel episode as source material.
3. Extract operational judgment: what they notice, ignore, act on, and invalidate.
4. Feed that back into the skill process when analyzing a live opportunity.
5. Prefer primary talks by the investor/operator over reaction content.

### Good prompts

```text
Using skills/buffett and the Buffett annual meeting clips as reference material,
evaluate whether this business is understandable, durable, and buyable with margin of safety.
```

```text
Using skills/dalio-principles-allweather and Dalio's Economic Machine video,
map the current growth/inflation regime and portfolio posture.
```

### Open source vs Questflow exclusive

These open-source skills + readings/podcasts/videos are free public judgment systems.

Questflow exclusive skills are closed-source systems distilled from top traders by labeling real-trade judgment. They are available to paid members at [questflow.ai](https://questflow.ai).
